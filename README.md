# odin-recipes
In this project, I will create a simple recipe website.

The website will consist of multiple webpages, and it will all be linked
back to the 'odin-recipes' repository on GitHub.

To accomplish this, several HTML skills will be used:
Writing boilerplate
Editing text to be bold or italic
Creating absolute links
Creating relative links
Including images
Making commits and pushes to GitHub

Enjoy exploring the finished project!
